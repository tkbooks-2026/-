# 《神明乩身》第二季 故事聖經 (Story Bible)

## 一、 核心概念
- **主軸**：第一季結束後，林天佑作為「多神明共用乩身」的名號不脛而走。上天開始下達更困難的任務，天佑因此決定籌設自己的「宮廟」來幫助更多人。
- **特色**：延續第一季的真實考究與熱血戰鬥，本季大幅擴展到 8 大傳統神明。同時融入「多女暗戀男主但發乎情止乎禮」的微後宮感情線，天佑依舊保持市井幽默與直男本色。
- **長度設定**：4 小時（共 120 集，每集 350-500 字）。分為八大單元，每單元 15 集。

## 二、 角色設定

### 1. 主角群
- **林天佑**：23歲。靈活機智，對神明極度虔誠但對感情異常遲鈍。
- **陳半仙**：老神棍兼前茅山道士，天佑的軍師與宮廟籌備主委。

### 2. 女角群 (微後宮感情線)
- **阿秀（青梅竹馬）**：隔壁麵攤女兒。總是在香鋪幫忙，煮麵給天佑吃。對天佑的感情最深，但覺得自己只是個普通女孩，不敢表白。
- **沈若冰（豪門千金）**：第一季城隍爺單元被救的富商之女。外表冰冷內心傲嬌，默默包辦了建廟的龐大資金，總是以「投資」為藉口接近天佑。
- **李清月（冷豔道姑）**：正統龍虎山傳人。原本不服天佑這個「野生乩身」，但在見識過他請神的威力後徹底折服，常常以「切磋道法」為由跑來香鋪。
*註：三女私下互相吃醋，但對天佑都抱持著「神明選中之人不可隨意褻瀆」的敬畏，導致感情線充滿笑料與拉扯。天佑則只把她們當好兄弟/金主/道友。*

## 三、 單元劇架構與神明考究

### 【第一單元：天上聖母 (媽祖) - 第 1-15 集】
- **神明考究**：林默娘提燈引航，海神慈悲。
- **劇情**：沿海村落接連發生離奇海難，水鬼「抓交替」。天佑請媽祖降駕。媽祖不以武力鎮壓，而是以慈悲之光點亮引路燈，超度百年怨靈。阿秀在此單元為天佑熬夜縫製護身符。

### 【第二單元：關聖帝君 (關公) - 第 16-30 集】
- **神明考究**：過五關斬六將，武神正氣，義薄雲天。
- **劇情**：地方黑道與邪教勾結，利用降頭術控制信徒吸金。天佑請關老爺降駕，青龍偃月刀劈開邪陣，展現極致的武力與浩然正氣。沈若冰的家族企業險些受害，對天佑更添仰慕。

### 【第三單元：文昌帝君 - 第 31-45 集】
- **神明考究**：掌管功名利祿，文房四寶皆可為法器。
- **劇情**：名校爆發集體歇斯底里，因極端升學壓力與忌妒心引來了強大的「筆仙」邪靈。文昌帝君降駕，以硃砂筆點破學生們的心魔，淨化校園。

### 【第四單元：註生娘娘 - 第 46-60 集】
- **神明考究**：保佑孕婦與新生兒，化解嬰靈怨氣。
- **劇情**：一個大家族因百年前重男輕女的溺嬰惡習，累積了龐大的嬰靈詛咒，導致家族斷子絕孫。註生娘娘降駕，展現極度溫柔卻又不可侵犯的神威，超度無辜靈魂。

### 【第五單元：濟公活佛 - 第 61-75 集】
- **神明考究**：降龍羅漢轉世，鞋兒破帽兒破，瘋癲中藏有大智慧。
- **劇情**：天佑捲入一場複雜的「三世桃花劫」，連身邊的三位女角也被波及而陷入奇異的迷惘中。濟公活佛降駕，以破蒲扇和酒葫蘆，用瘋癲搞笑的方式化解了宿命的情劫。

### 【第六單元：太歲星君 (六十甲子神) - 第 76-90 集】
- **神明考究**：凶神太歲，犯太歲之禍，掌管流年氣運。
- **劇情**：一名邪惡老道士專門盜取他人氣運（也就是犯太歲者的運氣）來逆天改命。李清月在追捕過程中受重傷。天佑請太歲星君降駕，以流年命盤為陣法，將老道士的反噬打入萬劫不復。

### 【第七單元：伏魔大帝 (鍾馗) - 第 91-105 集】
- **神明考究**：豹頭環眼，鐵面無私，專抓惡鬼。
- **劇情**：適逢農曆七月鬼門大開，地府陣法鬆動，引發百鬼夜行入侵市區。天佑化身鍾馗，手持青鋒劍與斬妖印，帶領五鬼搬運，上演一場暢快淋漓的都市除魔大戰。

### 【第八單元：玉皇上帝 (天公) - 第 106-120 集】
- **神明考究**：萬神之主，統御三界。
- **劇情**：天佑的宮廟即將落成，卻引來了覬覦凡間香火的「阿修羅界魔王」率領群魔攻擊。天佑迎來終極天劫，在瀕死之際，玉皇上帝的神念短暫降臨。萬神朝禮，天雷蕩魔。最終天佑正式獲頒「玉旨」，成為真正的宮廟之主。三女齊聚慶祝，天佑依然不懂少女心，溫馨搞笑收尾。

## 四、 寫作規範與提示
- **每集字數**：350~500 字，純對白與動作描述，**不可有 Markdown 標記或特殊符號**。
- **風格**：主角天佑幽默直男，神明降駕時展現莊嚴神威。感情戲要有「女生暗中吃醋、天佑一臉茫然」的喜劇效果。
- **結尾懸念**：每集（除單元結局外）都要卡在最緊張的時刻。
